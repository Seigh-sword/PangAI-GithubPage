# Pang AI
Pang AI is a,
AI extension in penguinmod it has blocks needed for all the AI projects you guys will use for
I am excited for how many people I will be helping and the people who will be learning how to use AI components in a such early age, you can use this anyway you like with no prompt limits 

**Not only chatbots** you can also use it for random idea or name generators its not my ideas its your idea and creativity which is being used 
## All Blocks
| Block Name | Type | Description |
| :--- | :--- | :--- |
| **AI ready?** | Boolean | Returns true if the AI server is online. |
| **when AI connects** | Hat | Starts a script when the AI is ready. |
| **when AI disconnects** | Hat | Starts a script if the AI goes offline. |
| **set system log to [ ]** | Command | Changes the AI's personality. |
| **set model to ( )** | Command | Changes the AI's "brain" (OpenAI, etc). |
| **ask prompt [ ]** | Reporter | Gets a text answer from the AI. |
| **get url of image [ ]** | Reporter | Gets a link to an AI-generated image. |
| **set temperature to ( )** | Command | Changes how creative the AI is. |
| **set seed to ( )** | Command | Sets a fixed number for the AI's logic. |
| **temperature** | Reporter | Shows the current creativity setting. |
| **seed** | Reporter | Shows the current seed number. |
